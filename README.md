# LearnMate

## Problem Statement

Students often struggle to identify their current skill level and choose the right learning path for their career goals.

## Solution

LearnMate is an AI-powered learning assistant built using IBM watsonx Orchestrate.

## Features

- GitHub Profile Analysis
- LeetCode Progress Analysis
- AI Skill Assessment
- Personalized Learning Roadmap
- IBM watsonx Orchestrate Agents

## Tech Stack

- React
- Node.js
- Express
- IBM watsonx Orchestrate
- GitHub API
- LeetCode GraphQL API

## Architecture

Frontend → Express Backend → GitHub & LeetCode APIs → IBM watsonx Orchestrate → Personalized Roadmap
